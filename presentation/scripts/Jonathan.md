# Guion

Hola, mi nombre es Jonathan Sánchez Pérez y participé en el desarrollo de la Interfaz de Métodos Numéricos versión 3.3.0.

Mi rol principal dentro del proyecto fue desarrollar la estructura inicial de la aplicación y varios de los primeros métodos numéricos implementados. Entre ellos se encuentran el método de Bisección, el método de Punto Fijo, Mínimos Cuadrados Polinomiales y el del Trapecio. Además, colaboré en el desarrollo de funcionalidades generales de la interfaz, como el sistema de búsqueda de métodos y el manejo de distintos tipos de error.

Mientras hablo, se puede observar parte del código correspondiente al método de Bisección, Punto Fijo y Mínimos Cuadrados Polinomiales y el del Trapecio. Este método permite encontrar raíces de ecuaciones mediante la reducción progresiva de un intervalo. También se muestran algunas secciones de la interfaz gráfica desarrollada para facilitar la interacción con el usuario.

Durante el desarrollo realicé diversas pruebas utilizando funciones matemáticas conocidas para verificar la precisión de los resultados. Estas pruebas permitieron comprobar que los métodos convergían correctamente y que la interfaz mostraba la información de manera clara y organizada.

Una de las principales dificultades fue diseñar una estructura suficientemente flexible para soportar distintos métodos numéricos sin tener que rehacer la interfaz para cada uno de ellos. Para resolver este problema se implementó una arquitectura modular que facilitó la incorporación de nuevas funcionalidades a lo largo del proyecto.

Como recomendación, considero que es importante dedicar tiempo al diseño inicial de la estructura del software antes de comenzar a programar. Una buena planificación permite mantener el código organizado y facilita el trabajo colaborativo cuando el proyecto crece.

Muchas gracias.
