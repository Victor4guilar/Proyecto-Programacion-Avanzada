# Guion

Hola, mi nombre es Gabriel Beltrán Amezcua y participé en el desarrollo de la Interfaz de Métodos Numéricos versión 3.3.0.

Mi rol principal fue implementar diversos métodos numéricos relacionados con aproximaciones y ecuaciones diferenciales. Entre ellos se encuentran el método de la Secante, Diferencias Finitas, Interpolación Lineal de Newton y el método de Runge-Kutta.

En pantalla se muestran algunos fragmentos del código desarrollado para el método de Secante, Diferencias Finitas, Interpolación Lineal de Newton y el método de Runge-Kutta, siendo este último uno de los algoritmos más utilizados para resolver ecuaciones diferenciales debido a su mayor precisión respecto a métodos más simples como Euler.

Durante el desarrollo se realizaron pruebas comparando diferentes métodos para observar sus niveles de precisión y comportamiento. Estas pruebas permitieron verificar que los resultados obtenidos eran consistentes y que las gráficas representaban adecuadamente las aproximaciones calculadas.

Una de las dificultades encontradas fue gestionar la gran variedad de parámetros que requieren algunos métodos numéricos. Para resolver este problema se mejoró la organización de los formularios y se incorporaron validaciones que ayudan a reducir errores durante la captura de datos.

La principal lección aprendida fue que una interfaz clara e intuitiva puede ser tan importante como el algoritmo matemático en sí. Una buena experiencia de usuario facilita el aprendizaje y reduce significativamente los errores de operación.

Muchas gracias.
