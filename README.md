Fecha: 12 de marzo del 2026 
Semestre: 4to 

Descripción del proyecto: en este proyecto vamos a crear una interfaz en la cual se puedan
                          resolver diferentes tipos de funciones como, ecuaciones aritméticas,
                          integrales y ecuacones diferenciales; en la interfaz se encontrará
                          un buscador para los métodos numéricos, asi como un menú, el cual
                          estará divido en los tres tipos de funcones que se manejaran en la
                          interfaz, y al seleccionar un tipo de función se desplegarán los tipos
                          de métodos numéricos que esten integrados para ese tipo de función;
                          así mismo, cuando se seleccione un método se abrirá una pestaña en 
                          la cual ya se podrá utilizar el método, ingresando la fución, los 
                          intervalos y el error permitido, obteniendo ahí mismo los resultados.
