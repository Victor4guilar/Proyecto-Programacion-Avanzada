Descripción del título: se escogió ese titulo para proyecto, porque el proyecto consiste en relaizar una interfaz 
                        en lenguaje Python, en la cual, al ingresar te despliegue el título métodos numéricos, así
                        mismo, aparezca un buscador para métodos numéricos, también aparecerá un pequeño menú, en 
                        cuál se podrá escoger entre tipos de metodos numéricos, en los que se encontrarán tres tipos,
                        para resolver ecuaciones artiméticas, integrales y ecuaciones diferenciales, en estos al 
                        escoger un tipo función se desplegará una lista de métodos para resolver ese tipo de funciones;
                        cuando se escoja un método un numérico, se abrirá otra pestaña en la cual se podrá utilizar el
                        método numérico y asignar la función, sus intervalos, así como el error permitido, y también 
                        se mostrará el resultado.

Objetivos generales: nuestro objetivo general es que por medio de una interfaz se puedan resolver diversas funciones, 
                     con la implementación de un método numérico, y sea fácil de interactuar, asi como un ambiente 
                     agradable al usuario.

Objetivos específicos: nuestros objetivos específicos son que al ingresar se pueda por medio de un buscador encontrar
                       el método numérico que el usuario desee (siempre y cuando este implementado); también que la 
                       interfaz cuente con un menú en el cual sea fácil de encontrar un método numérico, en el cual 
                       solo se encuenttren los tres tipos de funciones que vamos a implementar, y que al entrar a un 
                       tipo de función se desplieguen los métodos que esten implementados para resolver ese tipo
                       de funciones.
